# A Short Workshop on Introduction to AI and Large Language Models (LLMs)

This repository hosts materials for the GIRLsmarts workshop. This workshop aims to provide an accessible introduction to Artificial Intelligence (AI) and Large Language Models (LLMs) and inspire a more diverse group of students to explore Machine Learning.

The module slides are available under: `website/slides/`. This website is built using [Quarto](https://quarto.org/) and you first need to install it if you want to build it locally.

The slide materials now also include starter notebooks for word prediction and search engine demos under `website/slides/`, alongside a `pyproject.toml` and `uv.lock` for the Python environment used in the workshop.
